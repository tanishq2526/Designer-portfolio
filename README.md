# 🎨 PrimePixels | Interactive Designer Portfolio

Welcome to the **PrimePixels Portfolio**! This is a highly creative, interactive portfolio built with React and Vite, designed from the ground up to mimic the look, feel, and professional workspace of Adobe Photoshop. 

Built by Tanishq, a passionate Graphic Designer and visual storyteller, this portfolio showcases a unique way of presenting creative work, experiences, and brand identity through a web-based "canvas".

---

## ✨ Features

- **🖼️ Photoshop Workspace UI**: A meticulously crafted interface mimicking a professional design software environment.
- **📑 Tabbed Navigation**: Seamlessly switch between different "open documents" like Profile, About, Experience, Projects, and Contact.
- **🎨 Dynamic Layer Panel**: An interactive right-side panel that acts as the core layer structure of the portfolio.
- **🌓 Dark/Light Mode**: Toggle between a sleek dark mode (default) and a crisp light mode workspace.
- **📱 Fully Responsive**: A fluid design that gracefully adapts to mobile, tablet, and desktop views without breaking the canvas experience.
- **⭐ Interactive Feedback**: A custom 5-star rating system built right into the interface with a visual all-time feedback report.
- **✍️ Stylized Typography**: Utilizes a mix of clean system fonts and elegant serifs (`Playfair Display`) for a premium editorial look.

---

## 🚀 Tech Stack

- **Framework**: React.js
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Custom Design System, Flexbox/Grid)
- **Icons**: Lucide React

## 🛠️ Getting Started

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/tanishq2526/Designer-portfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd myPortfolio
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Visit `http://localhost:5173` to see the magic happen!

---

## 📂 Project Structure

- `src/components/` - Contains all the modular React components (`TopNav`, `Toolbar`, `Workspace`, `Panels`).
- `src/index.css` - The global stylesheet containing the unified design system and mobile responsive queries.
- `src/assets/` - Image assets, including the profile imagery and project placeholders.
